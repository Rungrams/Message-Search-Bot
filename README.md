# Message-Search-Bot
A Telegram Bot for searching any channel messages from Inline by **[TOBOT UPDATE](https://t.me/tobot_update)**.

I made this for [𝐋𝐢𝐬𝐭𝐁𝐨𝐭](https://t.me/repo_robot). You can use this for something else. Edit according to your use.

We have to use Bot for Inline Search & Userbot for Searching in Channels. So both Bot & Userbot will work together.
